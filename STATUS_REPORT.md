- `GEMINI_API_KEY`

### 2️⃣ Enable Branch Protection (10 min)
Go to: https://github.com/lexichron-ai/SocratesOS/settings/branches

Follow: `.github/BRANCH_PROTECTION.md`
- Create rule for `main` (2 approvals required)
- Create rule for `develop` (1 approval required)

### 3️⃣ Push Code & Deploy (5 min)
```bash

```
